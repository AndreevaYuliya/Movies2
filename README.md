# Movies2 Project

A Spring Boot application for managing movies and directors.

## Features

-   CRUD for Directors
-   CRUD for Movies
-   DTO architecture
-   MockMvc tests
-   H2 test database support

## Structure

    src/main/java/com/movies2
    src/test/java/com/movies2

## How to run tests

    mvn test

## How to build

    mvn clean package
